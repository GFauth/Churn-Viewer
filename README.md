Este é o trabalho final da cadeira de Classificação e Pesqusa de Dados do 3º semestre da UFRGS.

Nele, nós nos aprofundamos nos métodos, estruturas e algoritmos estudados na disciplina e trabalhamos
especificamente no armazenamento e a manipulação de dados em arquivos com dados salvos no formato binário
com o objetivo de praticar a indexação, a pesquisa e a ordenação de dados.
